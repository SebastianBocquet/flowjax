FlowJAX
===============
FlowJAX is a package distributions, bijections and normalizing flows using 
`equinox <https://github.com/patrick-kidger/equinox/>`_ and 
`jax <https://github.com/google/jax/>`_.


Installation
------------------------
.. code-block:: bash

    pip install flowjax
